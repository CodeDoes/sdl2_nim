sdl2-nim 0.95 beta
==================

sdl2-nim is a wrapper of the SDL 2 library for the Nim language.

* SDL homepage: http://www.libsdl.org/
* Nim homepage: http://nim-lang.org/

You need to have SDL 2 dynamic libraries installed on your system.
